## 2. Data Collection

The Mod:
- Does NOT collect personal data
- Does NOT store user data on external servers
- Does NOT transmit user data to third-party services except Discord via official IPC

The Mod only sends activity information to the local Discord client using Discord IPC.

---

## 3. Data Sent to Discord

Depending on user configuration, the following information may be sent to Discord:

- Minecraft version
- World name (optional)
- Player name (optional)
- Custom status text
- Custom image asset keys
- Custom URLs (if buttons are enabled)

This data is transmitted locally to the Discord client using Discord’s IPC protocol.

No data is stored by LighSync Games.

---

## 4. Data Storage

The Mod stores configuration locally on the user's device using Minecraft Forge configuration files.

No remote database or external storage is used.

---

## 5. Third-Party Services

The Mod interacts with:
- Discord (via official Rich Presence IPC)
- Minecraft (local game environment)

Users are subject to the Privacy Policies of these services.

---

## 6. Children's Privacy

The Mod does not knowingly collect data from children under the age of 13.

Since no personal data is collected, no targeted processing occurs.

---

## 7. Security

The Mod:
- Does not transmit data over the internet directly
- Does not operate external servers
- Uses local IPC communication with Discord
